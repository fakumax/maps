## Getting Started

This repository has:

_Create a new Next App with Typescript_

```sh
yarn create next-app --typescript
```

_Add ChakraUI Typescript_

```sh
yarn add @chakra-ui/react @emotion/react@^11 @emotion/styled@^11 framer-motion@^5

```

_Add Dotenv_

```sh
yarn add dotenv
```

_Add Mapbox GL_

```sh
yarn add mapbox-gl
```
